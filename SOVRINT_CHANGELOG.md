Added:

- governance, integrity, and evidence interface guide;
- strict and read-only JSON profiles;
- research profile constants in TypeScript, Python, and Go;

- cross-language validation workflow.

Compatibility:

- upstream JSON-RPC protocol unchanged;
- existing SDK users unaffected unless they apply the SOVRINT helpers;
- existing package names and client/session APIs preserved.
